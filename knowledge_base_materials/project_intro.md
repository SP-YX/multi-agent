# 多智能体协作系统技术白皮书

## 1. 系统概述

多智能体协作系统（Multi-Agent Workbench）是一个基于 LangGraph 构建的模块化、可扩展的多智能体任务处理框架。系统通过状态机驱动多个专业化智能体有序协作，将复杂任务自动拆解、分配、执行、汇总，最终输出结构化结果。

## 2. 核心架构

- 编排层：LangGraph StateGraph 状态机，管理节点流转和数据传递
- 智能体层：五个专业化 Agent（规划/RAG/搜索/编码/汇总）
- 工具层：向量检索、联网搜索、代码执行沙箱
- 记忆层：会话窗口 + 摘要 + 持久化
- 接口层：CLI / FastAPI / Streamlit

## 3. Agent 职责

- PlanAgent：任务拆解与执行计划生成
- RAGAgent：知识库文档检索与引证回答
- SearchAgent：实时互联网信息搜索
- CoderAgent：Python 代码自动生成与沙箱执行
- SummaryAgent：多源信息汇总与最终答案生成

## 4. 工程设计原则

- 单一职责：每个 Agent 只负责一个领域
- 配置驱动：业务参数、提示词、模型名全部外置
- 可观测性：中间件体系提供性能监控、Token 统计、日志追踪
- 错误隔离：每个节点独立 try/except，单节点失败不影响全局
- 记忆持久化：按 session_id 独立存储对话历史，支持冷启动恢复

## 5. 适用场景

- 智能客服（RAG + 搜索双通道）
- 数据分析报告自动生成
- 技术方案调研（规划 → 搜索 → 汇总）
- 代码审查与自动修复
